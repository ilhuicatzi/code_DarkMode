# Dark Mode

Dark Mode is a web page that shows how dark mode can be implemented on a web page, using different technologies such as React, TypeScript, Vite, Nextjs, Astro and TailwindCSS.

## Technologies

- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- [Nextjs](https://nextjs.org/)
- [Astro](https://astro.build/)
- [TailwindCSS](https://tailwindcss.com/)
